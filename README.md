**RAG-агент с БД сайта neoflex**

Для работы нужны langchain и langgraph, причём langchain брать версии prebuilt==1.0.4, как как в последней версии есть баг, не позволяещий нормально  работать с либой.

Агент работает через FastAPI, а клиент к нему цепляется через прокси – телеграм-бота.

Работает так себе, так как ранжируются тексты не очень, но диалог с памятью поддерживается.

Для запуска потребуется собственный openrouter API ключ и ключ от telegram-бота.
